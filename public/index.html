Auth

Sign Up

  • URL
    • /api/auth/signUp
  • Method
    • POST
  • Headers
    • Content-Type : application/json
  • Request Body (Raw JSON)
    • name as string
    • email as string, must be unique
    • password as string, must be at least 6 characters
  • Response 
                    
            {
                "message": "Successfully Sign Up Account",
                "status": 201
            }

Sign In

  • URL

  • Method
    • POST
  • Headers
